# 🚀 Nexora SaaS

Hey 👋
This is **Nexora**, a simple SaaS tool I built to help small sellers manage their products and orders without needing a full website or complicated setup.

---

## 💡 Why I built this

A lot of small sellers (especially WhatsApp / local business owners) don’t have:

* a proper website
* any system to track orders
* a way to manage products easily

Everything happens manually — chats, notes, confusion 😅

So I thought…
👉 why not build a **simple dashboard** where they can manage everything in one place?

---

## 🧠 What Nexora does

### 👨‍💼 For Sellers:

* Add and manage products
* Track incoming orders
* Update order status (Pending → Shipped → Delivered)
* See total revenue

---

### 🛒 For Customers:

* Browse products
* Place orders easily
* Choose payment method:

  * UPI (direct to seller)
  * Cash on Delivery

---

## 💰 Business Model (Important)

This is NOT a marketplace like Amazon.

👉 Instead:

* Customers pay **directly to the seller**
* Sellers pay **₹199/month** to use this tool

So basically:

```text
Customer → Seller (product payment)
Seller → Nexora (subscription)
```

---

## 🔒 Current Stage

This is an **MVP (Minimum Viable Product)**

* Payment verification is manual (for now)
* Built mainly for testing and early feedback
* Will improve with automation (Razorpay APIs, webhooks, etc.)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SQLite
* Razorpay (for subscription/payment links)

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/AYUSHMAN_KAR/nexora-saas.git
cd nexora-saas
pip install -r requirements.txt
streamlit run app.py
```
